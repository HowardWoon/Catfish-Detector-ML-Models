# Catfish Detector | Group 7

Flask site for the WIA1006 catfish detector assignment, ready for local use or Vercel deployment.

## Run

```bash
python run_web.py
```

Open:

- `/` scanner
- `/notebook` notebook cells
- `/models` model table
- `/export` download bundle and report

## Deploy To Vercel

1. Push the repository to GitHub.
2. Import the repo into Vercel.
3. Use `app.py` as the Flask entrypoint and deploy.
4. Open `/export` to download the cached model bundle or the report PDF.
